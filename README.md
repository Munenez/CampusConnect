# 🎓 Campus Connect

Campus Connect is a web-based university management and communication system built using PHP and MySQL.  
It allows students and administrators to interact through announcements, events, and dashboard management.

---

## 🚀 Features

- 🔐 Secure login system (Admin & Student)
- 📢 Announcements management
- 📅 Events posting and viewing
- 🧑‍💼 Admin dashboard
- 🎓 Student dashboard
- 🗄️ MySQL database integration

---

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- XAMPP (Local Development)
